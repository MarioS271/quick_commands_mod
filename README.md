# quick-commands-mod (fabric)
### A minecraft mod with GUIs for easy access to useful commands.

<br>

### Current Features (v1.1):
- Set the time
- Set the weather
- Give yourself some effects
- Manage your items

### Languages:
- English
- German
- German (Austrian)
- [PARTIAL] Russian
- [PARTIAL] Ukrainian
