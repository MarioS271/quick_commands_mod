# QuickCommands

A fabricmc minecraft mod with GUIs for easy access to useful commands.

Using this mod, you are able to execute **/time**, **/weather**, **/effect** and player scaling commands easily,
all from a simple GUI.

### Current Features (v1.2):
- Set the time
- Set the weather
- Give yourself some effects
- Manage your player's scale

### Languages:
- English
- German
- German (Austrian)
- [Up to 1.21.4] Russian
- [Up to 1.21.4] Ukrainian
